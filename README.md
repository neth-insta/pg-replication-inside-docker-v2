### PostgreSQL Replication Inside Docker (the same server)

```shell
$ git clone https://github.com/neth-insta/pg-replication-inside-docker.git
$ cd pg-replication-inside-docker
$ docker-compose up -d

$ rm -rf pg-replication-inside-docker
```

```shell
$ docker-compose up -d

$ docker ps
$ docker volume ls
$ docker network ls
```
